# IPL-Dashboard-2008-2025
This is Ipl multiple dataset that is used and make relationship with help of power bi to make interactive dashboard

This project presents an interactive Power BI dashboard built to analyze historical data from the Indian Premier League spanning from 2008 to 2025. The dashboard is designed to provide comprehensive insights into team performance, player achievements, and overall tournament trends in a visually engaging manner.


Key Features
Identification of season winners and runner-up teams
Orange Cap analysis (top run scorers)
Purple Cap analysis (top wicket takers)
Season-level statistics including:
Total matches played
Number of teams and venues
Total fours and sixes
Player performance insights:
Runs scored
Wickets taken
Boundary counts (fours and sixes)
Interactive points table showing team rankings, wins, and losses
Dynamic filtering to analyze specific seasons and compare trends
Tools & Technologies Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures and KPIs

Objective

The main goal of this project is to transform raw IPL data into meaningful insights that help users understand:

Team consistency and performance trends
Impact of top players across seasons
Evolution of scoring patterns in the league

Conclusion

This dashboard provides a clear and interactive way to explore IPL data, enabling users to quickly derive insights into team dominance, player excellence, and match trends. It serves as a strong data analytics portfolio project demonstrating skills in data visualization, storytelling, and business intelligence.
